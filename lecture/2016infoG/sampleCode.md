# プログラミング入門 コピー用コード

## 練習3-5

```
整数 i
pinMode(5, "OUTPUT")

i ← 0
i < 10 の間，
  | digitalWrite(5 ,1)
  | delay(1500)
  | digitalWrite(5, 0)
  | delay(500)
  | i ← i + 1
を繰り返す
```

```
整数 red

red ← 255
red > 0 の間，
  | analogWrite(9, red)
  | delay(10)
  | red ← red - 1
を繰り返す
```
